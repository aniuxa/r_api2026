1  Práctica 4: Tablascon {kable} y {gt}”

En esta práctica aprenderemos a presentar resultados en tablas legibles, ordenadas y visualmente cuidadas, usando dos herramientas muy comunes en R:

  • {kable} (y {kableExtra}), ideal para tablas simples y rápidas.
  • {gt}, pensada para tablas más elaboradas y publicables.

El objetivo no es solo “que se vean bonitas”, sino comunicar mejor los datos.

1.2 Una tabla sin formato (punto de partida)

Antes de embellecer, veamos cómo luce una tabla “cruda”.

wdi_raw <- WDI(
  indicator = "SP.POP.TOTL",
  country = c("MEX", "BRA", "ARG", "COL", "CHL"),
  star = 2022,
  end = 2022
)

Esta tabla funciona, pero:

  • no tiene título,

  • los números no están formateados

  • no comunica contexto.

wdi_raw %>%

  kableExtra::kable()

1.5 Introducción a {gt}: una nueva lógica

gt() funciona de manera distinta:

  • primero se crea el objeto tabla,

  • luego se le aplican transformaciones visuales.

Empezamos con lo más básico.

wdi_raw %>%
  select(country, SP.POP.TOTL) %>%
  rename(
    country_code = country,
    Población = SP.POP.TOTL
  ) %>%
  gt()
country_code Population, total
Argentina 45407904
Brazil 210306415
Chile 19553036
Colombia 51737944
Mexico 128613117
 
tabla_gt <- wdi_raw %>%
  select(country,  SP.POP.TOTL) %>%
  rename(
    country_code = country,
    Población = SP.POP.TOTL
  ) %>%
  gt() %>%
  tab_header(
    title = "Población total en países seleccionados",
    subtitle = "Banco Mundial, 2022"
  ) %>%
  fmt_number(
    columns = Población,
    decimals = 0,
    use_seps = TRUE
  )

1.7.4 Tabla bonita con gt + banderas (fmt_flag)

fmt_flag() inserta banderas a partir de códigos ISO (2 o 3 letras).
Aquí usamos ISO2 (por ejemplo, MX, GT, UY).

tabla %>%
  filter(year%in%c(2000, 2010)) %>% 
  gt::gt() %>%
  gt::fmt_flag(columns = iso2c) %>%
  gt::cols_label(
    iso2c = "",
    country = "País",
    year = "Año (WDI)",
    pop_total = "Población total",
    pib_pc_usd = "PIB per cápita (USD)",
    gdp_real_growth = "Crecimiento PIB real (%)"
  ) %>%
  gt::fmt_number(columns = pop_total, decimals = 0, use_seps = TRUE) %>%
  gt::fmt_currency(columns = pib_pc_usd, currency = "USD", decimals = 0) %>%
  gt::fmt_number(columns = gdp_real_growth, decimals = 1) %>%
  gt::tab_header(
    title = "Indicadores socioeconómicos (WDI) y macro (FMI-WEO)",
    subtitle = "Ejemplo de tabla con banderas usando {gt::fmt_flag()}"
  ) %>%
  gt::tab_spanner(label = "WDI", columns = c(year, pop_total, pib_pc_usd)) %>%
  gt::tab_spanner(label = "FMI (WEO)", columns = c(gdp_real_growth)) %>%
  gt::cols_align(align = "left", columns = c(iso2c, country)) %>%
  gt::cols_align(align = "right", columns = c(pop_total, pib_pc_usd, gdp_real_growth)) %>%
  gt::opt_table_outline() %>%
  gt::opt_row_striping() -> tabla_final
